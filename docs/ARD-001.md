## 1. What Atlas must become

Not a portfolio. Not a PIKU marketing site. Not a GitHub mirror.

**Atlas is the public operating system of Jaskirat Singh** — two access modes over one evidence layer:

```
┌─────────────────────────────────────────────────────────┐
│  EVIDENCE LAYER (build-time, from 8 repos + gh API)     │
│  raw/ → repos.json → patterns.json → copy               │
└──────────────────────────┬──────────────────────────────┘
                           │
         ┌─────────────────┴─────────────────┐
         ▼                                   ▼
   /  TRUST MODE                         /os  ATLAS MODE
   (20 seconds)                          (10–30 minutes)
   Typography carries truth               Depth rewards curiosity
```

**Disagreement with earlier plan:** "Problems.app first" was premature. After full repo analysis, the flagship Explorer experience should be **`Operate.app`** — how you move from ambiguity to shipped system — because that is what every private repo proves (Relive security hardening, operator routing, defensive sync, parked tracks, Bot Lab). PIKU is one terminal node in that graph, not the hub.
