# Singh Atlas

Minimal personal site — inspired by cinematic dark portfolio design.

## Sections

| Nav | Who it's for |
|-----|----------------|
| **Hello** | Recruiters — name, proof, contact in one glance |
| **Work** | Founders — 6 evidence tiles, click for depth |
| **Mind** | Quirky — five beliefs from repo patterns |

## Run

```bash
npm install && npm run dev
# http://localhost:3456
```

## Build & deploy

```bash
npm run build        # → out/
npm run build:content  # split evidence JSON
```

Railway: Docker + nginx:8080
